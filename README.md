# ccna-final-practice
A complete CCNA Final Exam Simulation website (French version) that runs entirely in the browser! Features include:  - 125+ questions from the official CCNA 1 ITNv7 exam - 30-question random practice exams - 15-minute time limit - Answer feedback (green = correct, red = wrong) - Username leaderboard - No backend required - just static HTML/CSS/JS
